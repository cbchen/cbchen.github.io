Privacy Policy

Last updated: April 29, 2026

This Privacy Policy describes how Four Fractals handles information in connection with its personal automation application, including its use of Google APIs for Gmail and Google Calendar access.

1. Overview

Four Fractals is a personal automation application used to help the app owner access, summarize, organize, and act on information from their own Google account, such as Gmail messages and Google Calendar events. The application is not intended for public use, advertising, resale of data, or third-party data brokerage.

2. Information the Application May Access

Depending on the permissions granted by the user, the application may access the following types of Google user data:

  • Gmail message metadata, such as sender, recipient, subject, date, labels, and thread identifiers.
  • Gmail message content, such as snippets or message bodies, when needed for email triage, summarization, or deadline extraction.
  • Google Calendar event information, such as event title, time, location, attendees, and descriptions.
  • Basic Google account information required for authentication and authorization.

3. How Information Is Used

Google user data is used only to provide personal automation features requested by the user, including:

  • Synchronizing recent emails for local review and processing.
  • Classifying emails for triage, summaries, reminders, or follow-up actions.
  • Extracting possible deadlines, tasks, or calendar event candidates from emails.
  • Creating, reading, or updating calendar events when explicitly requested or configured by the user.
  • Maintaining local processing state to avoid duplicate processing.

4. Data Storage

Data may be stored locally on the user's own computer or infrastructure for the purpose of running the automation workflow. This may include email metadata, selected email content, summaries, processing state, and calendar-related information. The application does not operate a public database of user information.

5. Data Sharing

Four Fractals does not sell Google user data. Four Fractals does not transfer Google user data to advertising platforms, data brokers, information resellers, or other third parties for advertising, profiling, credit-worthiness, or unrelated purposes.

Data may be sent to service providers only when necessary to provide the requested functionality, such as an AI model provider selected and configured by the user to summarize or classify email content. Where local-only processing is configured, data remains on the user's own machine except for communication with Google APIs.

6. Google API Limited Use Disclosure

Four Fractals' use and transfer of information received from Google APIs will adhere to the Google API Services User Data Policy, including the Limited Use requirements.

7. Data Retention and Deletion

Locally stored data is retained only as long as needed for the user's automation workflow. The user may delete local application data, logs, cached email content, summaries, and stored OAuth credentials at any time from the systems where the application is installed.

Revoking the application's Google access can be done from the user's Google Account security settings. Once access is revoked, the application will no longer be able to access Google user data unless the user reauthorizes it.

8. Security

Reasonable safeguards are used to protect stored data and credentials, including limiting access to the user's local machine or chosen infrastructure. However, no method of electronic storage or transmission is completely secure.

9. Children

The application is not directed to children under 13 and is not intended to collect information from children.

10. Changes to This Policy

This Privacy Policy may be updated from time to time. If the application's data practices materially change, this page will be updated before the new practices are used.
